# 🎨 Sistema de Diseño BEO

Bienvenido al sistema de diseño para los componentes web del Banco de Exportación de Colombia.

## 🚀 Características

- **Multi-Marca**: Soporta BEO y EBF con temas completamente separados.
- **Sincronización Automática**: Los estilos se actualizan automáticamente desde `src/styles/`.
- **Componentes Web**: Construido con Lit y Tailwind CSS.

## 🛠️ Instalación

```bash
npm install
npm run storybook
```
